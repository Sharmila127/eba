Community(group) Chat Socket Events

those events are make realtime communication between client and server

Join events

join events for each clients (student, teacher and admin) to join community(group) chat rooms.

Server side: socket.on("join",(data)=>{write your code})

Client side: socket.emmit("join",data)

data:{groupId,userId}

Send Message Event

student, teacher and admin to use this event for realtime transfer the message in group chat

Server Side: socket.on("sendmessage",(data)=>{write your code})

Client side: socket.emit("sendmessage",data)

data: {groupId, userId, message}

Recive Messsage Event

each client are recived the previous and current message to used this event

Server Side: io.to(groupId).emit("message",{message})

Client Side: socket.on("message",{message})

{message} recived sender message and transfer emit by groupId
